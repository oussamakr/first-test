HTML (hypertext markup language)

Cree votre site web avec html5 et CSS3

HTML signifi "hyperText markup language " qu'on peut traduit par " langaue des balises pour l'hypertexte".Il est utilisé afin de creer et représenterle contenu d'une page web et sa structure. D'autres technologies sont utilisees avec HTML pour decrire la présenation d'une page( CSS ) et / ou ses fonctionnalités interactives ( JavaScript).

HTML fonctionne grace a des "balises" qui sont inserees au sien d'un text normal. chacune de ces balises indique la signification de telle ou telle portion de texte dans le site. On parle d'"hypetexte" en reference aux liens qui connectent les pages web entre elles. C'est la mécanique originelle du "World wide Web " que nous connaissons aujourd'hui. en ecrivant et publiant des pages web,vous devenez un acteur du web des que votre site est accessible en ligne .
